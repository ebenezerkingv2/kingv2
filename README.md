# King

King's Full-Stack Portfolio

## Project Overview

This repository contains King’s personal portfolio, showcasing frontend and backend projects, skills, and live demos. The portfolio is built using modern web technologies in a full-stack setup.

## Structure

- `frontend` - Frontend application (React, Vite, TypeScript, TailwindCSS)
- `backend` - Backend API (Node.js, Express, MongoDB)

## Features

- Responsive design for mobile, tablet, and desktop
- Project showcase with live demo and GitHub links
- Animated UI using Framer Motion

## Setup Instructions

1. Clone the repo: `git clone ...`
2. Install frontend dependencies: `cd frontend && npm install`
3. Install backend dependencies: `cd backend && npm install`
